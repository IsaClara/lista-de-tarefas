# Lista de Tarefas – Space To-Do List
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)


Uma aplicação web de **lista de tarefas com pastas**, desenvolvida com **HTML, CSS e JavaScript puro**, com foco em organização, persistência de dados e uma experiência visual divertida inspirada no espaço.

---

##  Visão Geral

Este projeto permite que o usuário organize suas tarefas em **pastas personalizadas**, mantendo todas as informações salvas no navegador através do **LocalStorage**.  
Cada pasta possui sua própria lista de tarefas, que podem ser marcadas como concluídas ou removidas.

O projeto foi desenvolvido como parte do meu aprendizado em **desenvolvimento front-end**, com foco em manipulação do DOM, lógica em JavaScript e boas práticas de estruturação.

---

## Funcionalidades

- Criação e exclusão de pastas
- Listas de tarefas independentes para cada pasta
- Marcar tarefas como concluídas
- Remover tarefas individualmente
- Salvamento automático usando LocalStorage
- Interface estilizada com tema espacial
- Animações suaves e partículas decorativas
- Layout responsivo

---

## Tecnologias Utilizadas

- **HTML5** — estrutura do projeto  
- **CSS3** — estilização, animações e responsividade  
- **JavaScript (Vanilla JS)** — lógica, manipulação do DOM e armazenamento local  

---

## Conceitos Aplicados

- Manipulação do DOM
- Eventos (`click`)
- Estruturas de dados em JavaScript (objetos e arrays)
- Persistência de dados com `localStorage`
- Separação de responsabilidades (estrutura, estilo e lógica)
- UX básica (feedbacks visuais e alertas)
